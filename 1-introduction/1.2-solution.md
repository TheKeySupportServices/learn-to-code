# 1.2 Solution: Draw a rectangle with easy to set height and width

```python
import turtle
t = turtle.Turtle()

height = 200
width = 100

t.forward(width)
t.right(90)
t.forward(height)
t.right(90)
t.forward(width)
t.right(90)
t.forward(height)
t.right(90)
```
