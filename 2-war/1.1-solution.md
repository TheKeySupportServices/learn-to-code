# 1.1 Solution: Draw a square using Turtle!

```python
import turtle
t = turtle.Turtle()

t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
```

The final `t.right(90)` is a nicety which puts the Turtle back into its starting
position.
